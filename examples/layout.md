## Text layout possibilities:

#### h4
h4 elements are used as the titles of events
```html
<h4>TEXT</h4>
```
#### to create one block of text or an imagelist:
```html
<div class="frame">
  ...
</div>
```

#### italic
to get *italic* text:
```html
<i>TEXT</i>
```

#### bold
to get **bold** text:
```html
<b>TEXT</b>
```
